Common Fixes
403 Error (API Key rejected)
Go to Google Cloud Console → Credentials → your API Key → Edit.
Under "Application restrictions" set to None. Under "API restrictions" make sure Google Sheets API is allowed.
404 Error (Sheet not found)
Double-check your Sheet ID — it's the part of the URL between /d/ and /edit.
Also confirm the sheet is shared publicly: Share → Anyone with the link → Viewer.
Data loads but shows 0s
Your sheet headers must be lowercase with underscores exactly as shown (e.g. live_price not Live Price).
Tab names must be exactly: Income, Expenses, Assets, NetWorth (capital first letter, no spaces).
Numbers have currency symbols or commas (€1,234.00)
Enter numbers as plain numbers in your sheet: 1234.00 not €1,234.00.
The app strips $ and , automatically but not € — format cells as "Number" not "Currency" in Google Sheets.
How to format cells correctly in Google Sheets
Select all data columns → Format → Number → Number (not Currency, not Automatic).
This ensures values come through as plain numbers via the API.
